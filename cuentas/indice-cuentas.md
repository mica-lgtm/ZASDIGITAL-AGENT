## Regla de uso

Antes de crear campañas, descripciones, PDPs, landings, automatizaciones, creativos o desarrollos para una marca, el agente debe leer la ficha correspondiente en esta carpeta.

Si la marca no aparece en este índice, no debe tratarla como cuenta activa.

Si Mica menciona una marca que no está en este índice, el agente debe preguntar si se trata de:

1. Una cuenta nueva.
2. Una cuenta anterior/inactiva.
3. Una referencia externa.
4. Un error de nombre.

No debe inventar información de marcas que no estén registradas.
