# Negativas base (compartidas entre marcas)

Lista de partida para revisar contra los search terms de cualquier marca (además de las negativas propias declaradas en la ficha de cada marca). No aplicar ninguna sin verificar contexto -- algunas marcas sí quieren captar alguno de estos términos.

- "gratis" / "free" (salvo que la marca tenga una promo real de envío/producto gratis vigente)
- "usado" / "segunda mano" (para marcas que solo venden producto nuevo)
- "trabajo" / "empleo" / "cv" (términos de búsqueda de empleo, no de compra)
- "tutorial" / "como hacer" / "diy" (intención informativa, no transaccional)
- Nombres de marcas competidoras (salvo que la marca pida explícitamente competir por esos términos)
- "reclamo" / "denuncia" / "estafa" (intención de queja, no de compra)
